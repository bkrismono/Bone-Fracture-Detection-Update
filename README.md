# Bone-Fracture-Detection-Update
Bone Fracture Detection
The aim of this study was to develop and evaluate an explainable deep learning-based bone fracture classification system using ROIs from the multimodal HBFMID dataset, with exploratory analysis of probability calibration to improve reliability and interpretability in pre-clinical diagnosis. This dataset comprises 1,920 images and 5,842 ROI samples extracted from 10 fracture classes, with an imbalanced class distribution. The data was split using a patient-level strategy (70/15/15 for training, validation, and testing) to prevent data leakage, and model performance was evaluated through repeated experiments and statistical validation. The proposed methodology includes: (1) converting fracture detection annotations from the HBFMID dataset into an ROI classification dataset with a controlled truncation strategy; (2) training seven leading deep learning models including ResNet50, DenseNet121, EfficientNet-B3, ConvNeXt-Base, MobileNetV3-Large, ViT-Base, and Swin-Base using a standardized training pipeline with data augmentation and early stopping; (3) applying post-hoc temperature scaling to calibrate model output probabilities; and (4) visualizing model interpretability with Grad-CAM and Integrated Gradients to ensure transparency in clinical decision-making.
